# 教务系统日历文件导出系统

## 我的收获

- 学会了 TypeScript 静态类型编程，实践了 ES6 Promise、async、Module 特性
- 如何经济地挑选合适的第三方开源库，npm 包管理的方法

## 存在的不足

缺乏统一的设计规范。网页字体、段落排版、色彩搭配还任欠妥

## 期望的改进

可以把格式转化导出日历文件的逻辑代码抽取为库，使用户界面和算法模型分离开来。
